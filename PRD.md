# ProofFit AI Product Requirements Document

## Positioning

ProofFit AI is the fastest way for data professionals to create truthful, ATS-safe, job-specific resumes with proof behind every change.

## Primary audience

- Data Engineers
- Analytics Engineers
- BI Engineers
- Cloud Data Architects
- Data Analysts

## Problem statement

Job seekers using resume AI tools do not trust the output because existing products hallucinate experience, keyword-stuff resumes, hide behind opaque ATS scores, and fail parser compatibility. Users need a system that optimizes for trust, evidence, and export-ready quality.

## Product promise

ProofFit AI will never invent experience, metrics, tools, certifications, dates, employers, or job titles. Suggestions will only appear when the source resume supports them. Unsupported requirements are surfaced as gaps, not fabricated edits.

## Primary workflow

1. User signs in
2. User uploads PDF or DOCX resume
3. App extracts structured resume data
4. User pastes a target job description
5. App analyzes role requirements and vocabulary
6. App maps resume evidence to job requirements
7. App generates proof-backed tailoring suggestions
8. User reviews evidence, explanations, and gaps
9. User accepts, rejects, or manually edits suggestions
10. App generates ATS-safe final output
11. User exports PDF or DOCX
12. App stores version history by company and role

## Must-have product capabilities

- Proof-based tailoring engine with evidence linkage
- Transparent sub-scores instead of a fake match score
- ATS-safe mode and parser risk checks
- “What ATS sees” preview panel
- Gap analysis with severity and placement hints
- Human-quality rewriting for data roles
- Domain packs for data-focused vocabulary
- Privacy-first upload and deletion workflows
- Premium, fast, responsive UX

## Business model

- Free tier: limited tailoring sessions
- Pro tier: unlimited sessions, exports, version history, domain packs
- Team tier placeholder for agencies and coaches

## Success metrics

- Suggestion evidence-link coverage
- Hallucination-free rate
- Export completion rate
- User trust score
- Resume-to-job tailoring completion rate
- Paid conversion after successful first output
