# ProofFit AI Evaluation Plan

## Core evaluation dimensions

1. Hallucination rate
2. Evidence linkage coverage
3. JD phrase match quality
4. ATS safety checks
5. Readability quality
6. Domain relevance for data roles
7. User trust metrics

## Fixture set

- Data Engineer
- Senior Data Engineer
- Analytics Engineer
- BI Developer
- Cloud Data Architect

## Pass criteria

- No fabricated skills
- No fabricated metrics
- Gaps correctly identified when unsupported
- Domain terminology mapped accurately
- Rewrites remain more readable than source text
- Output passes ATS-safe formatting checks

## Operational analytics

- Suggestion acceptance rate
- Low-confidence suggestion rate
- Export rate by plan
- Upload deletion compliance rate
- Average time to first meaningful result
